Welcome to **Tim Fashion Academy**, a modern and dynamic web platform crafted with Next.js to showcase a premier fashion design and tailoring institution. This project offers an engaging user experience, highlighting courses, mentors, student impact, and a creative gallery, all designed to inspire the next generation of fashion entrepreneurs.

## Overview

This project is a sophisticated, responsive marketing website for a fashion academy, built on **Next.js 15**, **React 19**, and styled with **Tailwind CSS 4**. It leverages **Framer Motion** for captivating animations and **React Parallax Tilt** for interactive UI elements, providing a smooth and immersive browsing experience.

## Features

*   **Immersive Hero Section**: Dynamic introduction with engaging animations and a clear call to action.
*   **Academy Difference Showcase**: Highlights unique aspects of the academy using interactive 3D tilt cards.
*   **Craft Section**: Details the core skills taught, such as Precision Tailoring, Creative Design, and Color Mastery.
*   **Meet the Mentors**: Introduces the expert faculty with profile cards and social media links.
*   **Impact Metrics**: Visually appealing display of student success rates, projects, and trainees.
*   **Course Catalog**: Presents a clear overview of fashion design, pattern drafting, styling, and digital fashion programs.
*   **Creative Gallery**: Showcases student work and academy life with an interactive image lightbox.
*   **Call to Action (CTA)**: Encourages prospective students to enroll with prominent, animated buttons.
*   **Contact Form**: Seamless integration with Proforms.top for inquiries.
*   **Responsive Navigation**: Modern navigation bar that adapts for desktop and mobile devices.
*   **SEO-Friendly Structure**: Built with Next.js for optimal search engine performance.
*   **Performance Optimized**: Utilizes Next.js features for fast loading times and efficient asset handling.

## Usage

To get the Tim Fashion Academy website up and running locally, follow these simple steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/tim-fashion-academy.git
    cd tim-fashion-academy
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```
    This will start the application in development mode, typically accessible at `http://localhost:3000`.

4.  **Build for production**:
    ```bash
    npm run build
    # or
    yarn build
    # or
    pnpm build
    ```
    This command compiles the application for production deployment.

5.  **Start the production server**:
    ```bash
    npm run start
    # or
    yarn start
    # or
    pnpm start
    ```
    After building, this command serves the optimized production build.
